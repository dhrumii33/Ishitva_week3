# Week 3 – Git & Project Structure

## Goal
Learn Git basics, branching, merging, rebase, .gitignore, submodules.

## Folder Structure
/src       → source code  
/include   → header files  
/build     → cmake build folder  
/docs      → documentation  

## How to Build (CMake)
1. mkdir build  
2. cd build  
3. cmake ..  
4. make  

## Features
- Git branching  
- Merge conflict practice  
- Rebase practice  
- .gitignore  
- Submodule example
